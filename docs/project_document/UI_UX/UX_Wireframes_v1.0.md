## 10. Annotations and Notes

### 10.1 Global Annotations

| ID | Element | Notes |
|----|---------|-------|
| A1 | Header | Sticky on scroll, 64px height desktop, 56px mobile |
| A2 | Navigation | Dropdown menus on hover (desktop), tap (mobile) |
| A3 | Hero | Full-width, min-height 600px desktop, 400px mobile |
| A4 | Cards | 16px padding, 8px border radius, subtle shadow |
| A5 | Forms | Labels above inputs, 8px margin between fields |
| A6 | Buttons | 44px min touch target, 8px padding |
| A7 | Footer | Dark background, light text, 4-column layout |
| A8 | Sidebar | 280px width desktop, full-width overlay mobile |
| A9 | Tables | Horizontal scroll on mobile, sticky header |
| A10 | Modals | Centered, max-width 500px, overlay backdrop |

### 10.2 Interaction Notes

| Interaction | Desktop | Mobile |
|-------------|---------|--------|
| Navigation dropdown | Hover to open | Tap to expand |
| Form validation | Real-time, inline errors | Same |
| Data tables | Click row to expand | Tap, swipe for actions |
| Image galleries | Lightbox on click | Full-screen swipe |
| Date pickers | Calendar popup | Native date input |
| Tooltips | Hover | Long press |
| Context menus | Right-click | Long press |

### 10.3 Responsive Behavior

| Breakpoint | Layout Changes |
|------------|----------------|
| < 640px | Single column, hamburger menu, bottom nav for portals |
| 640-768px | Two columns, compact header |
| 768-1024px | Sidebar collapsible, 3-column grids |
| 1024-1280px | Full sidebar, 4-column grids |
| > 1280px | Max container width, centered content |

### 10.4 Animation Guidelines

| Element | Animation | Duration |
|---------|-----------|----------|
| Page transitions | Fade in | 200ms |
| Modal open | Scale + fade | 200ms |
| Dropdown open | Slide down | 150ms |
| Hover effects | Color transition | 150ms |
| Loading states | Skeleton shimmer | Infinite |
| Success feedback | Check mark bounce | 300ms |
