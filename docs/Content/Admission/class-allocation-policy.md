# Class Allocation Policy

## Student Placement Guidelines

**Document Version:** 1.0
**Last Updated:** January 11, 2026

---

## Allocation Principles

Students placed in classes based on:
- Age-appropriate grade
- Previous educational level
- Assessment results
- Special considerations

## Maximum Class Size

25 students per class maximum

## Criteria

**Primary Factors:**
- Age requirements
- Previous grade completed
- Academic readiness
- Social development

**Special Considerations:**
- Learning needs
- Language proficiency
- Maturity level
- Parent consultation
