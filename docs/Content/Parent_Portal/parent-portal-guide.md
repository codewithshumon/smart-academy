## Welcome to the Parent Portal

## Getting Started

### Accessing the Portal

**Portal URL:** [portal.smartacademy.edu.bd](https://portal.smartacademy.edu.bd)

**System Requirements:**
- Internet connection (broadband recommended)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Device: Computer, tablet, or smartphone
- JavaScript enabled

### First-Time Login

**Step 1: Receive Your Credentials**
- Parents receive login credentials upon enrollment
- Username: Typically parent email address
- Temporary password: Provided by school
- Keep credentials secure and confidential

**Step 2: Initial Login**
1. Go to portal URL
2. Enter your username and temporary password
3. You'll be prompted to create your own password
4. Create a strong password (minimum 8 characters, include letters and numbers)
5. Select a security question and answer
6. Accept the Terms of Use
7. Click "Complete Setup"

**Step 3: Complete Your Profile**
- Add your contact information
- Upload profile photo (optional)
- Set communication preferences
- Add student information (if not pre-loaded)

### Forgot Password?

**Reset Your Password:**
1. Click "Forgot Password" on login page
2. Enter your registered email address
3. Check email for reset link
4. Follow instructions to create new password
5. Login with new password

**Troubleshooting:**
- If reset email doesn't arrive within 5 minutes, check spam folder
- Contact school IT support if problems persist
- IT Support: it.support@smartacademy.edu.bd

## Troubleshooting

### Common Issues

**Issue: Cannot Login**
- Check username and password
- Clear browser cache and cookies
- Try different browser
- Reset password if forgotten
- Contact IT support if problem persists

**Issue: Portal Not Loading**
- Check internet connection
- Clear browser cache
- Disable browser extensions
- Try different browser
- Check if portal is under maintenance

**Issue: Cannot View Information**
- Ensure pop-up blockers are disabled
- Enable JavaScript in browser
- Update browser to latest version
- Contact school if data appears incorrect

**Issue: Payment Failure**
- Check payment details are correct
- Ensure sufficient funds/balance
- Try different payment method
- Contact accounts office for assistance
