## About Robotics Club

### Mission

> "To develop technical skills, creativity, and problem-solving abilities through robotics, preparing students to be innovators who use technology for beneficial purposes."

### Philosophy

**Technology as Trust (Amanah)**:
- Technology is a tool from Allah
- Must be used for beneficial purposes
- Knowledge brings responsibility
- Innovation should serve humanity
- Ethical considerations in technology

**Muslim Contributions to Science**:
- Algebra and algorithms (Al-Khwarizmi)
- Engineering marvels of Islamic civilization
- Scientific method development
- Preserving and advancing knowledge

### Club Details

**Eligibility**: Class 5-9
**Meeting**: Tuesday and Thursday, 3:30-5:00 PM
**Location**: Innovation Lab (Room 310)

## What We Do

### Core Activities

**Robot Design and Building**:
- Learn mechanical design principles
- Build robots using kits and custom parts
- Understand robot components and systems
- Apply engineering concepts

**Programming and Coding**:
- Learn programming languages (Python, Scratch, C++)
- Code robot behaviors and movements
- Develop algorithms for tasks
- Debug and optimize code

**Project-Based Learning**:
- Design solutions to real problems
- Build prototypes
- Test and iterate designs
- Document and present projects

### Weekly Schedule

**Tuesday**: Building and Design
- Mechanical design principles
- Construction techniques
- Component assembly
- Testing mechanical systems

**Thursday**: Programming and Testing
- Coding fundamentals
- Robot programming
- Sensor integration
- Project testing

## Competitions and Projects

### Competitions

**Regional Robotics Competition**:
- Annual event in Dhaka
- Project-based challenges
- Robot games and races
- Innovation showcase

**Categories**:
- **Line Follower**: Robot follows line using sensors
- **Maze Solving**: Navigate complex paths
- **Sumo Wrestling**: Push opponent out of ring
- **Innovation**: Original robot projects
- **Programming Challenge**: Complete tasks autonomously

### Student Projects

**Recent Projects**:
- **Automatic Prayer Time Alert System**: Uses GPS to calculate prayer times
- **Smart Water Dispenser**: Conserves water while making wudu easier
- **Quran Holder Automation**: Opens and closes Quran pages with buttons
- **Solar-Powered Automatic Door**: Energy-efficient entry system

## Team Structure

### Competition Teams

**Roles**:
- **Project Manager**: Coordinates team efforts
- **Programmer**: Develops code
- **Builder**: Constructs robot
- **Designer**: Plans robot appearance and structure
- **Tester**: Evaluates performance

**Team Requirements**:
- 3-5 students per team
- Mix of skills and strengths
- Commitment to full competition season
- Regular attendance at meetings
