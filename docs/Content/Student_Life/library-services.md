## Library Services

### Circulation

**Borrowing Privileges**:
- **PlayGroup-KG**: Classroom use only
- **Class 1-3**: 1 book for 1 week
- **Class 4-5**: 2 books for 1 week
- **Class 6-9**: 3 books for 2 weeks

**Checkout Process**:
- Present book at circulation desk
- Show student ID
- Book stamped with due date
- Return book on or before due date

**Renewals**:
- Items may be renewed if not reserved
- Maximum 2 renewals per item
- Bring book to library for renewal

**Fines**:
- Overdue: 2 taka per day per book
- Lost or damaged: Replacement cost
- Fines may be waived for good cause

### Reference Services

**Research Help**:
- Assistance finding information
- Guidance on research strategies
- Help using library catalog
- Instruction on using reference materials
- Bibliographic instruction

### Technology Services

**Available**:
- Computer stations for research
- Internet access for educational purposes
- Printing (for school work)
- Photocopying (when appropriate)
- Wi-Fi for personal devices (educational use)

## Library Policies

### Behavior Expectations

**In the Library**:
- Use quiet voices
- Walk, don't run
- Handle books with care
- Return books to proper place
- Clean up workspace
- Respect others' right to read

**Book Care**:
- Clean hands when handling books
- No writing in books
- Turn pages carefully
- Use bookmarks (no dog-earing)
- Protect books from damage

### Conduct

**Appropriate Use**:
- Reading and studying
- Research and homework
- Quiet collaboration
- Respectful conversation

**Not Allowed**:
- Food or drinks (water in closed container OK)
- Cell phone use
- Loud talking or disruptive behavior
- Games unrelated to academics
- Damaging library materials

### Internet Use

**Acceptable Use**:
- Research and homework
- Educational websites
- Online catalog searching
- Teacher-approved activities

**Not Allowed**:
- Social media
- Games
- Inappropriate content
- Downloading without permission
